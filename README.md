## partner client

**partner**，一个私人生活辅助系统，前后端分离。本仓库是partner项目的前端部分，后端仓库地址：[partner](https://github.com/yinyanlv/partner) 

## 主要技术

**前端：** angular6+，material design

**后端：** rust，actix-web，diesel，mysql，redis
